# Guía uso

Para acceder y trabajar en los distintos módulos se utilizan las "ramas" o branch.

Se pueden ver las ramas que existen y en la que nos encontramos con `git branch`

Para moverse entre ramas se utiliza `git checkout <nombre_rama>`

Dentro de github, podemos ver las ramas desde la interfaz gráfica del sitio web o en el apartado "[Branches](https://github.com/dmadcham/BigData/branches/all)"
